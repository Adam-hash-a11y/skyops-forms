# skyops-forms

A frontend project built with Vite for practicing forms, props, and component patterns in React.

## Tech Stack

- Vite
- React

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

```bash
git clone https://github.com/your-username/skyops-forms.git
cd skyops-forms
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
